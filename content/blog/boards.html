When building an home automation system you need to face different questions like security, availability, connectivity and interoperability between the others. In the previous article we pointed out how great is to experiment and hack devices as they are cheap, but those solutions are not (yet) enough powerful to feature all we need for a connected device in the house. What if we want to add SSL, off-line control, protocols interoperation? We have two solutions: single device intelligence and home automation hubs.


Single Device Intelligence

This is the best solution and it totally change the way we think at home automation today. Every device comes out with the ability to connect to the web in a secure way. The scenario is the one where you go to Ikea or the shop down to your house, buy a light, go home, connect it to the web and tell Lelylan it's part of your house.

This sounds great right. Yes, it does but it has a couple of drawback. The first one is that every device must be equipped with the ability to directly connect to the web (or to the router in your house) providing all security features (SSL and authorization logics). This is possible and today we already have the technologies to do this, but it would make any device way more expensive.

Think at a light. Today you pay it $10. What if it suddenly costs you more than $70 just to be connected to the web? No way, at least not if we think that home automation is for everybody. The good point is that techs are becoming more cheap and more powerful day after day and this will let us build smart devices for few dollars more really soon.


Home Automation Hub

This is the real solution nowadays. Think at the home automation ecosystem today. We have lot of companies producing any kind of device using different standards. In this way we depend from them and we can't exploit the real power of an home automation system because we do not let every device communicate.

To solve this problem we need an hub, which is a small computer that will let your house talk to the web, without caring at all about the technology devices uses. In this way you can connect devices that use Zigbee, Konnex, X10 and whatever exists out there and let them accessible to the web. Its not easy, as you need to have a great knowledge of how those protocols work out, but once you do it they are accessible.

An home automation hub gives you several advantages. You have only one place where to handle secure communication and authorization, you can install some extra applications, you can easily handle an off-line situation when internet does not work and much more.


Board Computers Compared

We think that making every single device smart is the future path, but in the meantime we need to face reality. For made a little research listing the boards out there that could be a reasonable solution to build your next home automation hub.

Raspberry PI

This is probably the best solution out there. It's a real computer where you can run Debian Linux at only $25. The initiative, started at Cambridge University got lot of attention from all media around the world and in 4 months they were able to sell over 500.000 pieces.

Price Raspberry PI | $25
Raspberry Model B | $35
OS Debian Linux. See other distributions.
Connectivity The base version does not offer connectivity.
Raspberry Model B gives you an Ethernet access.
Pros Low price, community, documentation, open-source.
Cons Computational power, wireless connectivity.

Rikomagic MK802

A fully functional computer no bigger than a USB pen drive, the Rikomagic MK802 runs on the popular Android 4 operating system and it features Wifi. The design is just cool and you have a fully functional computer no bigger than a USB pen drive.

Price MK802 | $74
OS Android 4.
Connectivity Wifi 802.11b/g/n.
Pros Low price, wireless connectivity, design.
Cons Missing community, documentation, computational power.

Mele A1000

Available for as little as $70, while that's twice the price of a Raspberry Pi computer, the Mele A1000 includes a faster processor, more RAM and a case. It can be a better choice if the power given by the Raspberry PI is not enough and you need something aesthetically nice.

Price Mele A1000 | $70
OS Android and Ubuntu.
Connectivity Ethernet by default.
Pros Low price.
Cons Missing community, wireless connectivity, computational power.

BeagleBoard

The BeagleBoard is an low cost, high performance, low power OMAP3 based platform designed by BeagleBoard.org community members. It is open-source and there are hundreds of applications developed on top of it.

Price BeagleBone | $89
BeagleBoard xM | $149
OS Android, and major Linux distributions like Ubuntu, Debian, ArchLinux and Gentoo. See supported distributions
Connectivity Ethernet by default.
Pros Experience, community, documentation, computational power, open-source.
Cons Price, wireless connectivity.

PandaBoard ES

PandaBoard is a low-cost, open OMAP 4 mobile software development platform.

Price PandaBoard ES | $189
OS Android, Ubuntu and Linaro. See supported distributions
Connectivity Ethernet connectivity and Wifi 802.11 b/g/n.
Pros Community, computational power, wireless connectivity, focus on mobile.
Cons Price.

Plug Computers (Sheevaplug)

Main resource for the plug computer development community. Plugcomputer.org offers solutions built on a Marvell system-on-chip and focus on high performances and giving full connected devices readily available for developers.

Price GuruPlug Display | $199
D2Plug | $249
OS Ubuntu, FreeBSD, NetBSD, Windows CE and any other operating system that is compatible with ARM CPUs.
Connectivity Ethernet connectivity by default. Model D2Plug offers integrated 802.11 b/g/n Wifi.
Pros Commercial support, computational power, design, wireless connectivity.
Cons Missing community, closed-source, price.

Ionics Plug

Plug computer with main focus on connectivity and low price. Ionos, comes from the same family of Sheevaplug computers, but with lower prices and better features. For sure one of the best solutions whenever you want to use a Plug Computer and you are searching for advanced wireless capabilities (WifI, Zigbee and Z-Wave ).

Price Nimbus 100 | $99
Stratus | $149
OS Debian 5 Lenny.
Connectivity Ethernet connectivity by default. Stratus model offers integrated 802.11 b/g (WPS support on Client Mode) and ZigBee. You can decide to change the Zigbee module for a Z-wave module.
Pros Commercial support, computational power, design, price, wireless connectivity.
Cons Missing community, closed-source.
